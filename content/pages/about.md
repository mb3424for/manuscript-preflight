Manuscript Preflight is built around a simple premise: authors should not have to trust memory alone for the final mechanical inspection of a scholarly document.

The free tool checks repeatable features of a DOCX—citation-reference consistency, comments, tracked changes, temporary markers, metadata, anonymity risks, heading structure, and selected journal requirements. It does not decide whether an argument is original, whether a method is valid, or whether a journal should accept the paper.

## Design principles

### Privacy before convenience

The free scanner performs its work locally in the browser rather than requiring the user to upload an unpublished manuscript.

### Flags rather than silent rewriting

Scholarly documents contain ambiguity. The tool therefore identifies possible problems and explains what to inspect instead of automatically changing citations or intellectual content.

### Searchable guidance around the tool

Each focused guide explains a concrete submission problem in ordinary language. The website is intended to be useful even before a user runs the scanner.

### Progressive verification

Future paid features may add external bibliographic verification and human review, but these should remain visibly distinct from local deterministic checks.

## Independent status

Manuscript Preflight is an independent project and is not affiliated with or endorsed by any journal, publisher, citation-style organization, university, or manuscript-submission platform.
