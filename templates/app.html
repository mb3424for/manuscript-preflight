What this manuscript checker does—and does not do
It performs a repeatable document audit. It does not replace journal instructions, expert copyediting, reference verification against scholarly databases, or human judgment about ambiguous citations.
Citation consistency
Compares common author-year citation keys against the detected reference list and flags likely mismatches.
File hygiene
Inspects DOCX internals for comments, tracked changes, temporary markers, and author-identifying core properties.
Journal profile
Checks the abstract length, keyword range, and required section headings you supply.


Frequently asked questions
Is my manuscript uploaded?
No. In the current free version, the DOCX is processed by JavaScript in your browser.
Does the tool correct citations automatically?
No. It reports likely mismatches because automatic scholarly citation corrections can introduce new errors.
Which citation styles are supported?
The current heuristic is most useful for common author-date formats. Numbered citation styles require a later parser.
